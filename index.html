hw-03: Interactive Webpage, and Marks & Channels

Marks

  1. Points (states on map)
  2. Area (bar)

Channels

  • Color (both map and bar)
    • Represents the possible categories for political majority and the candidates representing them (red=Trump/Republican majority, blue=Clinton/Democrat majority, red/blue stripes=tie/swing state).
    • This is a good channel to use as it makes the data quick to interpret by using preexisting color associations and following standard design practice for similar political visualizations of election data.
  • Shape (map only)
    • Represents the actual shape of the state that each point is presenting election data for.
    • This is a good channel to use as it makes the data quick to interpret by using preexisting shape associations and following standard design practice for similar political visualizations of election data.
  • Position (Both, map only)
    • Represents the geospatial location of the state that each point is presenting election data for.
    • This is a good channel to use as it makes the data quick to interpret by mapping data onto a recognizable geospatial plane (the United States) and following standard design practice for similar political visualizations of election data.
  • Size (Length, bar only)
    • Represents the proportion of points (electoral college votes) encompassed/held by each category (candidate/political party).
    • This is a good channel to use as it simiplifies the data below into visualization where the proportion and total majority is much easier to discern, even resembling the political "tug of war" of the election through a recognizable visual language.

Colormap(s)

  • Categorical
    • Blue represents a Democratic majority (Clinton) and red represents a Republican majority (Trump).
    • This is a great use of color as it takes advantage of preexisting color associations where blue is understood to represent the political left (liberal) and red is understood to represent the political right (conservative) in the American two-party political system. Red and blue also have high contrast making this visualization accessible for readers that may be colorblind.

Marks

  1. Points (bins)

Channels

  • Color (both map and bar)
    • Represents to what degree above/below average the scoring value of each bin (batted balls) is based on linear weights (LWTS).
    • This is a good use of this channel as the divergent colormap allows for hits that are well above or well below average stand out to the reader and provides enough contrast to identify trend-based groupings and discern the two extremes from one another.
  • Position (Both, Horizontal)
    • Represents the speed off the bat or "exit velocity" of each bin (batted balls) in miles per hour (mph) from 40 to 120.
    • This is a good use of this channel as it produces a 2D plane in which viewers can observe the relationship between two attributes, launch angle and exit velocity, and how these factors may contribute to the scoring value of a given hit. Exit velocity data is also a linear attribute, so presenting it on an axis like this makes it easy to interpret.
  • Position (Both, Vertical)
    • Represents the launch angle of each bin (batted balls) in degrees from -50 to 50.
    • This is a good use of this channel as it produces a 2D plane in which viewers can observe the relationship between two attributes, launch angle and exit velocity, and how these factors may contribute to the scoring value of a given hit. Launch angle data is also a linear attribute, so presenting it on an axis like this makes it easy to interpret. The addition of additional labels on this axis ("Ground balls", "Line drives", "Fly balls", and "Pop-ups") adds further grouping into specific value ranges that help provide a bit more context between this data and the game of baseball.

Colormap(s)

  • Divergent
    • Increasing saturation of orange represents a higher degree above average scoring value (LWTS) while increasing saturation of purple represents a higher degree below average scoring value. No saturation (white) represents the average scoring value.
    • This is a good use of color as it allows for hits that are well above or well below average to visually stand out to the reader and provides enough contrast to identify trend-based groupings and discern the two extremes from one another. One recommendation, however, would be to use more meaningful colors to communicate the score-based nature of this channel (say green as above average and red as below average) though this may reduce accessibility for viewers that are colorblind.
